# lod-shacl-light
Limited implementation of [W3C SHACL validation](https://www.w3.org/TR/shacl/)

## Supported targets

- sh:targetClass

## Supported paths

- sh:path

## Supported constraints

- Cardinality: sh:minCount, sh:maxCount
- String-based: sh:minLength, sh:maxLength, sh:languageIn, sh:uniqueLang
- Other: sh:hasValue

## Other

- sh:deactivated
